## Execute Media Scripts (vv) from remote

- **Note:** [`Node.js`](https://nodejs.org/en), [`ffmpeg`](https://ffmpeg.org) and [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) required in PATH.

```sh
npx -y github:Florencea/my-macos-build
```
